You can change the values of Excel cells, slicers or pivot table filters during the automation of Excel templates.
If you create a **Parameter Change** automation step, you can choose any of the available parameters.

For each [parameter](Excel-selection-parameters-for-automation) that should be changed you can then set the required value. If a parameter support multiple values they can as well be configured here.

<img alt="parameter changes in Excel automation using AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-automation-step-parameter-change-01-autotoolit-601.png')} width="601" className="dropShadowImages"/>

For each [parameter](Excel-selection-parameters-for-automation) an input help is available, if the parameter is supporting it.
