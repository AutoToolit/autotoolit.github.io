You can use parameters to make changes in your Excel template through automation or from PowerPoint.

### Define a new parameter
To define a new parameter start by selecting an Excel cell:

<img alt="select Excel cells for parameter selection in AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-01-cell-selection-automation-autotoolit-311.png')} width="311" className="dropShadowImages"/>

Then click the **Add Parameter** button in the **Auto** ribbon toolbar:

<img alt="add parameter for Excel automation in AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-02-add-automation-autotoolit-65.png')} width="65" className="dropShadowImages"/>

### Parameter properties in side panel
This will create a new parameter and show its settings in the **Automation** side panel:

<img alt="Excel automation parameter property definition using side panel in AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-03-side-panel-automation-autotoolit-341.png')} width="341" className="dropShadowImages"/>

### Linked object types

You can choose, what type of object the parameter should change:
- values in Excel cells
- selections in Excel slicers or
- filters in Pivot Tables

<img alt="elements available for automation in Excel using AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-04-source-type-automation-autotoolit-309.png')} width="309" className="dropShadowImages"/>

### Input help
If you choose to change Excel cells, you can also define the possible source values that should be selectable for the user by selecting the corresponding Excel cells:

<img alt="select input help cells for Excel automation parameters using AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-05-input-help-key-cells-automation-autotoolit-356.png')} width="356" className="dropShadowImages"/>

After selecting the Excel cells click the Excel cell selection button in the side bar to use those cell values as master data for the parameter:

<img alt="assign input help for Excel automation parameters using AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-06-input-help-key-setting-automation-autotoolit-313.png')} width="313" className="dropShadowImages"/>

In case the displayed texts for selection should be different from the key value applied to the parameter you can choose as well corresponding cells for text values:

<img alt="select text value cells for Excel automation parameters in AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-07-input-help-text-cells-automation-autotoolit-356.png')} width="356" className="dropShadowImages"/>

Click the Excel cell selection button to apply the selected cells to the parameter:

<img alt="assign key value cells for Excel automation parameters in AutoToolit for Office" src={useBaseUrl('/img/uiExcel/excel-parameter-08-input-help-text-setting-automation-autotoolit-313.png')} width="313" className="dropShadowImages"/>

The definition of input help values for parameters is optional. In case no input help is defined, any parameter value can be entered in the parameter.
Also the separate definition of key and text values is optional. In case no separate values are defined, the text values will be applied as key as well.

You can see the list of available parameters when clicking on the back button:
